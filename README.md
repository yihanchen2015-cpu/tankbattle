# 坦克大战（修复版）

纯前端 Canvas 坦克对战游戏，无需安装依赖。

## 运行

先完整解压 ZIP，再双击解压目录中的 `index.html`；也可以使用任意静态文件服务器打开目录。不要只从压缩包预览器中单独打开 HTML，否则浏览器可能找不到同目录的样式和脚本。

## 操作

- WASD / 方向键：移动
- 鼠标：瞄准
- F：射击
- T：切换主炮、机枪和高射炮
- G：终极技能
- Y：自动瞄准
- V：切换 2D 俯视 / 3D 战术视角
- 牛牛直升机：E / 空格上升，Q / Shift 下降；触屏设备使用高度按钮

## 可用模式

经典模式、守点模式、绝地偷袭、夺旗模式、感染模式、风暴模式。

PvP 联机和 WebRTC 好友房间需要服务器及信令服务，本离线版本将它们明确标记为开发中。

## 地图与武器特性

- 沙漠：大视野、障碍少，随机沙尘暴会降低视野并干扰直升机飞行。
- 雪地：留下履带轨迹；久停会冻结引擎、减速并掉血，移动后逐渐恢复。
- 城市：规则街区、十字路口、道路标线和多层建筑组成真正的城市巷战地图。
- 海岛：7 座独立据点岛由桥网连接。普通坦克可以入水，但会大幅减速并持续掉血；多多号步战车免疫进水伤害，水中速度为 30%。
- 机枪不能攻击直升机；主炮对直升机伤害为 40%。高射炮升空后在 600 范围内只锁定一次，最多制导 0.85 秒，转向能力受限。
- 可选 2D 俯视或 Three.js 真 3D 战场；3D 模式使用锁定地图北方、只平移不旋转的战术相机，并提供 WebAudio 音效、敌我名称牌、伤害飘字、据点进度和直升机边框预警；直升机与高射弹拥有真实 Z 坐标。
- 红方 AI 拥有更大的战场感知范围、目标集火、基地维修补给、毒圈转移和更积极的基地进攻逻辑。无有效目标时 AI 会停火并保持当前炮塔朝向。
- APS 仅有 4 次拦截，每次拦截后冷却 15 秒。牛牛直升机在每张地图都有对应的环境负面效果。
- 直升机拥有可控飞行高度；高度不足时会撞上建筑或障碍，6 秒内连续撞击 4 次将起火并持续受损。
- 风暴模式毒圈外基础伤害为 32 点/秒，离安全区越远伤害越高。
- 游戏按实际触摸事件和触控能力显示摇杆，与屏幕分辨率无关。
- 阵亡精彩回放使用独立 Three.js 3D 导演镜头：跟随致命炮弹前方、接近时拉远、0.3× 慢放命中，最后环绕展示车体解体。
- 采用团队积分制：普通击杀 +250、直升机 +300、占点 +500、夺旗 +1,000、摧毁基地 +10,000,000；玩家和 AI 队友都会为全队计分，结算按总分判定胜负。
- HUD 下方的公告栏实时播报击杀、占点、夺旗与基地事件；结束后使用全屏黑场胜负界面和蓝红 VS 积分板，可返回主页或使用上一局配置再来一局。
- WebAudio 使用多层音色合成引擎轰鸣、主炮低沉炮声、机枪连射、高射炮啸声与命中/击杀爆炸，并联动 2D/3D 镜头震动。
