# 坦克大战（修复版）

纯前端 Canvas 坦克对战游戏，无需安装依赖。

## 运行

先完整解压 ZIP，再双击解压目录中的 `index.html`；也可以使用任意静态文件服务器打开目录。不要只从压缩包预览器中单独打开 HTML，否则浏览器可能找不到同目录的样式和脚本。

## 操作

- WASD / 方向键：移动
- 鼠标：瞄准
- F：射击
- T：切换主炮、机枪和高射炮
- G：终极技能
- Y：自动瞄准

## 可用模式

经典模式、守点模式、绝地偷袭、夺旗模式、感染模式、风暴模式。

PvP 联机和 WebRTC 好友房间需要服务器及信令服务，本离线版本将它们明确标记为开发中。

## 地图与武器特性

- 沙漠：大视野、障碍少，随机沙尘暴会降低视野并干扰直升机飞行。
- 雪地：留下履带轨迹；久停会冻结引擎、减速并掉血，移动后逐渐恢复。
- 城市：规则街区、十字路口、道路标线和多层建筑组成真正的城市巷战地图。
- 海岛：7 座独立据点岛由桥网连接。普通坦克可以入水，但会大幅减速并持续掉血；多多号步战车免疫进水伤害，水中速度为 30%。
- 机枪不能攻击直升机；主炮对直升机伤害为 40%；高射炮具有伪 3D 弹道、可越障，升空后会追踪最近的坦克或直升机。
- 红方 AI 拥有更大的战场感知范围、目标集火、基地维修补给、毒圈转移和更积极的基地进攻逻辑。无有效目标时 AI 会停火并保持当前炮塔朝向。
- APS 仅有 4 次拦截，每次拦截后冷却 15 秒。牛牛直升机在每张地图都有对应的环境负面效果。
- 风暴模式毒圈外基础伤害为 32 点/秒，离安全区越远伤害越高。
